**Turn Proxy** — специализированный Android-клиент от **Torvalds VPN** для обхода блокировок и маршрутизации трафика (проброс WireGuard через TURN-серверы), разработанный с акцентом на автоматическую настройку, отказоустойчивость в зонах строгих ограничений и поддержку облачных хранилищ для зашифрованных конфигураций.

> [!IMPORTANT]
> Для полноценной работы приложения и успешного прохождения авторизации требуется **активная подписка** в сервисе Torvalds VPN.

### Текущий статус проекта
В данный момент репозиторий находится в процессе активного развертывания и рефакторинга. 

### В ближайшее время здесь появится:

* Детальный Роадмап (Roadmap) развития проекта.
